# ZeroTrace Admin Frontend (Vite + React + TypeScript)

Ye frontend **ZeroTrace backend** (Express + Mongo + WS) ke saath match karta hai.
- REST APIs: `/api/*` + `/admin/*`
- WebSocket (admin): `/ws/admin`
- UI pages: Login → Dashboard → Devices → Device Detail → SMS → Forms/Payments → Favorites → Sessions → Crashes → Settings

---

## ✅ Requirements

- Node.js **v18+**
- Backend running on `http://localhost:3000` (default)
